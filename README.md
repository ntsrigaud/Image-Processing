# Image Processing

This repository contains laboratory exercises, homework assignments, and extensive notes covering theoretical and practical concepts in digital image processing.

## Repository Structure

### Labs
Practical implementations of image processing techniques:
- **`Arithmetic-Operations/`**: Basic arithmetic operations for image blending, masking, and noise reduction.
- **`Bit-Planes/`**: Slicing techniques for image analysis, compression, and watermarking.
- **`DCT/`**: Discrete Cosine Transform applications for frequency analysis and image compression.
- **`Dithering/`**: Techniques for simulating continuous color depth on restricted palettes.
- **`Floyd-Steinberg-Error-Diffusion/`**: In-depth implementation of spatial dithering and error diffusion.
- **`Histograms/`**: Generation, equalization, and manipulation of image histograms.
- **`Outlier-Method/`**: Spatial filtering for noise removal while preserving edge integrity.
- **`Uniform-Quantization/`**: Analysis of continuous pixel intensity mapping to discrete quantization levels.

### Homework Assignments
Assessments evaluating foundational and advanced knowledge:
- **`Grayscale-Conversion/`**: Digital image representations and basic spatial operations.
- **`Histogram-Equalization/`**: Spatial filtering, intensity transformations, and histogram processing.
- **`Neighborhood-Processing/`**: Frequency domain filtering, Fourier transforms, and restoration techniques.

### Course Notes
Theoretical foundations aligned with the curriculum, located in the **`Notes/`** directory:
- **`Notes/01/`**: Foundations of digital image representation and applications.
- **`Notes/02/`**: Spatial domain operations, pixel relationships, and geometric transformations.
- **`Notes/03/`**: Intensity transformations and spatial filtering.
- **`Notes/04/`**: Frequency domain filtering and the Fourier transform.
- **`Notes/05/`**: Image restoration, reconstruction, and noise modeling.
- **`Notes/Research/`**: Supplementary research notes and advanced algorithmic readings.

### Research Projects
- **`CleanDIFT/`**: Advanced research project detailing diffusion-based image processing benchmarks and architectures.

## Objective

This repository serves to bridge theoretical image processing concepts with practical engineering, offering a comprehensive collection of resources for learning spatial and frequency domain image manipulation.
